

## 指针

``` cpp
int a[] = {1, 2, 3, 4};
int x = 5;
int *pointer1 = a;
int *pointer2 = &x;
```


``` cpp
char a[] = "abcdefg";
int *pointer1 = a;  // wrong
char *pointer2 = a; // correct
```


## 引用

``` cpp
int a = 10;
int &ref = a;  // ref 是 a 的引用
ref = 30;      // 通过引用修改 a 的值
std::cout << "a = " << a << std::endl;  // 输出: a = 30
```
